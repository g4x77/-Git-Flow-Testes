# Comandos Git e GitFlow

## Clonar repositório
git clone URL

## Status
git status

## Adicionar arquivos
git add .

## Commit
git commit -m "mensagem"

## Push
git push origin main

## Criar branch
git checkout -b nome-da-branch

## Trocar branch
git checkout nome-da-branch

## Corrigir commit
git commit --amend

## Atualizar
git pull

## Merge
git merge nome-da-branch
